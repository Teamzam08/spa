# TO-DO LIST (SPA)

This single-page to-do application features a
fluid user interface that– by using JavaScript–
allows users to rapidly add dynamic content.

Skills used:
HTML,
CSS,
RUBY,
ALGORITHMS, 
JAVASCRIPT

Purpose:
Create a to-do list and cross out tasks once they are complete.
